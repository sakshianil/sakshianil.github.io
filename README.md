# Dr. Sakshi Bharti Portfolio

Static GitHub Pages portfolio for Dr. Sakshi Bharti.

## Structure

- `index.html` - main bilingual portfolio page.
- `styles.css` - minimal monochrome site styling.
- `assets/images/` - visual assets used by the page.
- `assets/resumes/` - public CV PDFs.
- `assets/resumes/source/` - editable HTML sources used to regenerate the CV PDFs.

## Public CV Links

- English: `assets/resumes/dr-sakshi-bharti-cv-en.pdf`
- German: `assets/resumes/dr-sakshi-bharti-cv-de.pdf`

The root folder should stay small: only site entry files, license, README, and GitHub Pages config.
